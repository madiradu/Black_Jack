# Black_Jack
BlackJack game written in Typescript and Javascript for desktop use (not finished)<br>
For install on Windows run the next commands<br>
Install NodeJS<br>
npx nodegui-packer --init MyAppName<br>
cd MyAppName<br>
npm run build<br>
npx nodegui-packer --pack ./dist<br>
